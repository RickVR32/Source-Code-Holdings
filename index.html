What we build

Awareness.
Measured.

Source Code Holdings develops the Observer Code framework — a structured methodology for mapping individual and organisational behavioural patterns across four primary source codes. The output is measurable awareness data with direct applications in personal development, business intelligence, and AI-assisted coaching.

Framework · Live

Awareness Architecture

Proprietary four-axis behavioural mapping system. Classifies individual patterns across NARC, YARC, CIST, and OLL source codes. Produces a quantified Observer Code profile with Balance Scale scoring and awareness level assignment (1–10).

Outcome: measurable behavioural classification · Assessment platform live

Software · Live

Assessment Platform

Web-based psychometric assessment tool delivering individual Observer Code profiles. Free tier and paid full assessment (£19.99). Deployed via Riclee Digital. Business awareness assessment in development — digital footprint analysis and organisational pattern mapping.

Outcome: scalable digital assessment delivery · Active users

Hardware · TRL 2–3

Four devices.
In development.

Source Code Holdings is developing four hardware prototypes designed to passively capture environmental and biometric data for integration with the Awareness Architecture framework. All devices are at concept and early prototype stage (TRL 2–3). Full technical specifications available under NDA.

Device 01 · DP-1

Duality Processor

Dual-channel cognitive state monitor

Processes parallel input streams to identify cognitive state transitions and pattern drift in real time. Designed to integrate with the Observer Code scoring system.

TRL 2 Concept validated

Device 02 · ABFS

Ambient Field Sensor

Environmental awareness capture system

Passive environmental sensor array designed to detect and map ambient field conditions relevant to behavioural state classification within the Awareness Architecture.

TRL 2 Concept validated

Device 03 · NES

Neural Engagement System

Engagement pattern detection device

Detects and classifies neural engagement patterns, providing real-time input to the Awareness Architecture scoring engine. Target application: coaching, focus optimisation, awareness training.

TRL 2 Prototype stage

Device 04 · TCE

Terminal Consciousness Engine

Primary awareness computation unit

Central processing unit for the hardware ecosystem. Aggregates data from DP-1, ABFS, and NES to produce a unified real-time awareness score. The hardware brain of the system.

TRL 3 Early prototype
